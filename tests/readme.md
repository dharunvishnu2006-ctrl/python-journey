# 🎓 Student Grade Tracker

A Python web app to track student grades and rankings.

## Features
- Upload CSV file with student data
- Auto calculate grades (A/B/C/D/F)
- Rank students by marks
- Built with Streamlit

## How to Run
```bash
streamlit run app.py
```

## Tech Stack
- Python 3.14
- Streamlit
- pytest
